# @ai-rpg-engine/starter-ronin

> **रचना का उदाहरण:** यह शुरुआती पैकेज दिखाता है कि इंजन को सामंती रहस्य के लिए कैसे उपयोग किया जाता है। यह सीखने के लिए एक उदाहरण है, कॉपी करने के लिए कोई टेम्पलेट नहीं। अपना गेम बनाने के लिए [रचना गाइड](../../docs/handbook/57-composition-guide.md) देखें।

**जेड वेल:** एक सामंती किला, जो एक तनावपूर्ण राजनीतिक शिखर सम्मेलन के दौरान है। एक सरदार को ज़हर दिया गया है। हत्यारे को ढूंढें, इससे पहले कि सम्मान खत्म हो जाए।

[एआई आरपीजी इंजन](https://github.com/mcp-tool-shop-org/ai-rpg-engine) शुरुआती पैकेज कैटलॉग का हिस्सा।

## विषय

सामंती रहस्य + दरबार की साजिश। सम्मान नाजुक है - झूठे आरोप बहुत महंगा पड़ते हैं और उनसे उबरना लगभग असंभव है। हर सवाल का महत्व है, हर आरोप के परिणाम होते हैं। हत्यारे रोंइन को "एक ऐसा तलवार जो किसी सरदार का नहीं है - अप्रत्याशित" मानते हैं।

## त्वरित शुरुआत

```typescript
import { createGame } from '@ai-rpg-engine/starter-ronin';

const engine = createGame();
engine.start();
```

## प्रदर्शित पैटर्न

| विशेषता | रोंइन क्या दिखाता है |
|---------|------------------|
| **Engagement** | कई रक्षक भूमिकाएं (बॉडीगार्ड + समुराई), गुप्त रास्ते |
| **Resources** | द्वि-स्तरीय: कि (पुनर्जीवित होने वाला) बनाम सम्मान (नाजुक, ठीक करना मुश्किल) |
| **Social** | परिणामों के साथ जांच - झूठे आरोपों से सम्मान कम होता है |
| **Cognition** | हत्यारे की धारणा का नियम, जो किसी भी संबद्ध रोंइन को लक्षित करता है |

## सामग्री

- **5 क्षेत्र:** किले का द्वार, मुख्य हॉल, चाय का बगीचा, सरदार का कक्ष, गुप्त मार्ग
- **3 एनपीसी:** सरदार ताकेदा (ज़हर दिए गए सरदार), लेडी हिमिको (संदिग्ध), मजिस्ट्रेट सातो (जांचकर्ता)
- **2 दुश्मन:** शैडो हत्यारा, भ्रष्ट समुराई
- **1 संवाद वृक्ष:** मजिस्ट्रेट द्वारा ज़हर और दरबार के संदिग्धों पर जानकारी
- **1 प्रगति वृक्ष:** ब्लेड का मार्ग (स्थिर हाथ → आंतरिक शांति → धर्मी क्रोध)
- **1 वस्तु:** इत्र किट (5 कि को पुनर्स्थापित करता है)

## अद्वितीय यांत्रिकी

| क्रिया | विवरण |
|------|-------------|
| `duel` | अनुशासन का उपयोग करके औपचारिक मार्शल चुनौती |
| `meditate` | एक टर्न की कीमत पर कि और संयम को पुनर्स्थापित करें |

## आंकड़े और संसाधन

| आंकड़ा | भूमिका |
|------|------|
| अनुशासन | मार्शल कौशल, तलवार तकनीक, ध्यान |
| धारणा | जागरूकता, अनुमान, इरादे को समझना |
| संयम | सामाजिक नियंत्रण, भावनात्मक महारत |

| संसाधन | रेंज | टिप्पणियाँ |
|----------|-------|-------|
| HP | 0–30 | मानक स्वास्थ्य |
| सम्मान | 0–30 | नाजुक - झूठे आरोपों से -5, ठीक करना मुश्किल |
| Ki | 0–20 | आध्यात्मिक ऊर्जा, 2/टिक पर पुनर्जीवित होती है |

## क्या उधार लेना है

कई रक्षक भूमिकाएं (बॉडीगार्ड + समुराई) और द्वि-स्तरीय संसाधन (कि + सम्मान)। अध्ययन करें कि कैसे दो अलग-अलग ट्रिगर शर्तों वाली रक्षात्मक भूमिकाएं एक बहुस्तरीय रक्षा बनाती हैं, और कैसे कि (पुनर्जीवित होने वाला) बनाम सम्मान (नाजुक, ठीक करना मुश्किल) युद्ध और जांच में अलग-अलग खेलने की शैलियों को मजबूर करते हैं।
