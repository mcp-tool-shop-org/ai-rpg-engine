# @ai-rpg-engine/starter-colony

> **रचना का उदाहरण:** यह शुरुआती संस्करण विज्ञान-फाई कॉलोनी के अस्तित्व के लिए इंजन को कैसे कनेक्ट करना है, यह दर्शाता है। यह सीखने के लिए एक उदाहरण है, कॉपी करने के लिए कोई टेम्पलेट नहीं। अपना गेम बनाने के लिए [रचना गाइड](../../docs/handbook/57-composition-guide.md) देखें।

**सिग्नल का नुकसान:** एक दूरस्थ कॉलोनी पृथ्वी से संपर्क खो देती है। नीचे की गुफाओं में कुछ जीवित है।

[AI RPG इंजन](https://github.com/mcp-tool-shop-org/ai-rpg-engine) शुरुआती पैक कैटलॉग का हिस्सा।

## विषय

विज्ञान-फाई कॉलोनी प्रबंधन + एलियन संपर्क। ऊर्जा एक साझा कॉलोनी संसाधन है - जब यह कम होता है, तो सिस्टम एक के बाद एक विफल होने लगते हैं। एलियन उपस्थिति उपनिवेशवादियों को "विघटनकारी अनुनाद पैटर्न" के रूप में देखती है।

## त्वरित शुरुआत

```typescript
import { createGame } from '@ai-rpg-engine/starter-colony';

const engine = createGame();
engine.start();
```

## प्रदर्शित पैटर्न

| विशेषता | कॉलोनी क्या दिखाती है |
|---------|-------------------|
| **Engagement** | चोकपॉइंट ज़ोन टैग, दस्ते-आधारित बैकलाइन/सुरक्षा भूमिकाएं |
| **Resources** | पर्यावरण के कारण ऊर्जा का क्षरण होने वाला साझा कॉलोनी-व्यापी ऊर्जा संसाधन |
| **Environment** | ज़ोन के खतरे जो संसाधनों की कमी और सिस्टम विफलताओं को ट्रिगर करते हैं |
| **Cognition** | गैर-मानवीय धारणा नियम वाला एलियन इकाई |

## सामग्री

- **5 ज़ोन:** कमांड मॉड्यूल, हाइड्रोपोनिक्स बे, परिधि बाड़, सिग्नल टावर, एलियन गुफा
- **2 NPC:** डॉ. वास्केज़ (वैज्ञानिक), चीफ ओकाफोर (सुरक्षा)
- **2 दुश्मन:** हैक किया हुआ ड्रोन, अनुनाद इकाई
- **1 संवाद वृक्ष:** डॉ. वास्केज़ द्वारा एलियन सिग्नल और कॉलोनी की राजनीति पर जानकारी
- **1 प्रगति वृक्ष:** कमांडर पथ (फील्ड इंजीनियर → शार्प सेंसर → अनशेकेबल)
- **1 आइटम:** इमरजेंसी सेल (20 ऊर्जा बहाल करता है)

## अद्वितीय यांत्रिकी

| क्रिया | विवरण |
|------|-------------|
| `scan` | जागरूकता का उपयोग करके सेंसर स्कैन करें |
| `allocate` | कॉलोनी सिस्टमों के बीच ऊर्जा का पुनर्वितरण करें |

## आंकड़े और संसाधन

| आंकड़ा | भूमिका |
|------|------|
| इंजीनियरिंग | सिस्टमों को ठीक करें और बनाएं |
| कमांड | नेतृत्व और चालक दल का मनोबल |
| जागरूकता | सेंसर और धारणा |

| संसाधन | रेंज | टिप्पणियाँ |
|----------|-------|-------|
| HP | 0–25 | मानक स्वास्थ्य |
| ऊर्जा | 0–100 | साझा कॉलोनी संसाधन, 2/टिक पर पुनर्जीवित होता है |
| मनोबल | 0–30 | चालक दल का सामंजस्य |

## क्या उधार लेना है

पर्यावरण-संचालित संसाधन दबाव और दस्ते की भागीदारी भूमिकाएं। अध्ययन करें कि कॉलोनी का ऊर्जा संसाधन पर्यावरण संबंधी घटनाओं (केवल लड़ाई नहीं) के कारण कैसे कम होता है, जिससे सिस्टम विफलताओं की एक श्रृंखला शुरू होती है जो दस्ते में सामरिक संसाधन आवंटन को मजबूर करती है।
