# @ai-rpg-engine/starter-weird-west

**धूल के तूफान का सौदा** — एक सीमावर्ती शहर में एक ऐसा गुप्त पंथ छिपा हुआ है जो लाल चट्टानी इलाके से कुछ बुला रहा है।

[AI RPG इंजन](https://github.com/mcp-tool-shop-org/ai-rpg-engine) स्टार्टर पैक कैटलॉग का हिस्सा।

## विषय-वस्तु

पश्चिमी + अलौकिक। बंदूकधारी, धूल के भूत, और एक चट्टानी इलाके का पंथ। "धूल" नामक संसाधन समय के साथ जमा होता है — जब यह 100 तक पहुंचता है, तो यात्री रेगिस्तान का शिकार हो जाता है।

## शुरुआत कैसे करें

```typescript
import { createGame } from '@ai-rpg-engine/starter-weird-west';

const engine = createGame();
engine.start();
```

## सामग्री

- **5 क्षेत्र:** यात्री का चौराहा, बार, शेरिफ कार्यालय, लाल चट्टानी इलाके का मार्ग, आत्माओं का स्थान
- **2 गैर-खिलाड़ी पात्र (NPC):** बारटेंडर साइलास, शेरिफ हेले
- **2 दुश्मन:** धूल का भूत, चट्टानी इलाके का रेंगने वाला
- **1 संवाद वृक्ष:** बारटेंडर द्वारा दिया गया जानकारी जो चट्टानी इलाके के पंथ के बारे में है
- **1 प्रगति वृक्ष:** बंदूकधारी मार्ग (तेज़ हाथ → दृढ़ इच्छाशक्ति → अचूक निशाना)
- **1 वस्तु:** सेज बंडल (धूल को 20 से कम करता है)

## अद्वितीय विशेषताएं

| क्रिया | विवरण |
|------|-------------|
| `draw` | तेज़ गति से द्वंद्वयुद्ध — प्रतिक्रिया प्रतियोगिता |
| `commune` | ज्ञान का उपयोग करके आत्माओं से बात करें |

## आंकड़े और संसाधन

| आंकड़ा | भूमिका |
|------|------|
| दृढ़ता | सहनशक्ति और इच्छाशक्ति |
| तेजी | प्रतिक्रिया और प्रतिक्रिया समय |
| ज्ञान | अलौकिक ज्ञान |

| संसाधन | दायरा | टिप्पणियाँ |
|----------|-------|-------|
| HP | 0–30 | मानक स्वास्थ्य |
| दृढ़ संकल्प | 0–20 | मानसिक दृढ़ता, प्रति टिक 1 अंक की दर से पुनः उत्पन्न होता है |
| धूल | 0–100 | **विपरीत दबाव** — जमा होता है, 100 = मृत्यु |
